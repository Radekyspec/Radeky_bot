# Radeky_bot

### Runtime requirement
- Python version >= 3.9

## How to start

#### Clone this repository

```shell
git clone https://github.com/Radekyspec/Radeky_bot.git
```

#### Entering the directory of bot 

```shell
cd Radeky_bot
```

#### Install dependencies

```shell
pip install requirements.txt
```

#### Run your bot

```shell
nb run
```
